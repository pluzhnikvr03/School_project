Как работает наша система?
(Работа базы данных) 

Сценарий "ХОЧУ ВЗЯТЬ КНИГУ":
1. Пользователь сканирует QR-код
2. Бот вызывает get_book_info() → показываем информацию о книге
3. Бот вызывает is_book_available() → проверяем, свободна ли она
4. Если свободна → вызываем take_book()
5. Если занята → сообщаем "Книга уже выдана кому-то другому."

Сценарий "ХОЧУ ВЕРНУТЬ КНИГУ":
1. Пользователь сканирует QR-код
2. Бот вызывает user_has_book() → проверяем, его ли это книга
3. Если его → вызываем return_book()
4. Если не его → сообщаем "Эта книга не ваша".
