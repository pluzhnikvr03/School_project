Как работает наша система?
(Раюота базы данных) 

Сценарий "ХОЧУ ВЗЯТЬ КНИГУ":
1. Пользователь сканирует QR-код
2. Бот вызывает get_book_info() → показывает "Это книга 'Математика', автор Иванов"
3. Бот вызывает is_book_available() → проверяет, свободна ли
4. Если свободна → вызываем take_book()
5. Если занята → сообщаем "Книга уже выдана"

Сценарий "ХОЧУ ВЕРНУТЬ КНИГУ":
1. Пользователь сканирует QR-код
2. Бот вызывает user_has_book() → проверяет, его ли это книга
3. Если его → вызываем return_book()
4. Если не его → сообщаем "Эта книга не числится за вами"
